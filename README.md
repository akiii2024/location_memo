# Location Memo - フィールドワーク記録アプリ

フィールドワーク等野外調査の記録を効率化するメモアプリです。
画像として取り込んだ地図状にピンを置いてメモを作成します。ピンには文字だけでなく、写真や音声を添付することができます。

- マップ画面スクショ置き場
- メモ入力画面スクショ置き場

## できること

- 地図画像状にピンを配置し、メモを作成
- 複数枚の写真や音声メモを添付
- カテゴリ・発見者・標本番号等の詳細情報を付与
- AIによる画像の判別、音声文字起こし、メモ内容の改善提案
- PDFや画像ファイルとして出力、印刷

## アプリの使い方
1. 地図を追加

ログイン後、ホーム画面の「＋地図」から地図画像をアップロード

!地図一覧

2. ピンを配置してメモ作成

地図を開き、記録したい位置をタップ。タイトルや詳細情報、写真・音声を登録

!メモ作成

3. AIアシストやPDF出力を活用

写真から自動でメモ文案を生成したり、調査結果をPDFで出力することもできます

## 主な機能
 - 地図管理（複数の地図を切り替え）
 - レイヤー機能（１つの地図に複数のレイヤーを作成）
 - 情報つきメモ（カテゴリ・標本番号など様々な情報を添付可）
 - 画像/音声の添付、AIに夜メモ生成補助
 - 検索とフィルタ（地図別・カテゴリ別など）
 - 印刷機能（地図画像、ピンリスト、PDF出力）
 - バックアップと復元（データの安全な保存と復旧）

## システム要件

### 対応デバイス・OS

#### Android
- **OSバージョン**: Android 7.0（API 24）以上
- **インストール方法**: APKファイルまたはPWA（プログレッシブウェブアプリ）

#### iOS
- **インストール方法**: PWA（プログレッシブウェブアプリ）
- **アクセスURL**: https://akiii2024.github.io/location_memo/
- **備考**: 将来的にApp Storeでの配布を予定

#### Web（PC・タブレット）
- **対応ブラウザ**: Chrome、Safari、Firefox、Edge等の主要ブラウザ
- **アクセスURL**: https://akiii2024.github.io/location_memo/
- **PWA対応**: ブラウザからホーム画面にインストール可能

### 必要な機能・権限

- **カメラ**: 写真撮影・画像添付機能に必要
- **マイク**: 音声メモ録音機能に必要
- **ストレージ**: 地図画像・写真・音声ファイルの保存に必要
- **インターネット接続**: 
  - 基本的な機能（地図表示・メモ作成・編集）はオフラインでも利用可能
  - クラウド同期・AI機能・Firebase認証にはインターネット接続が必要

### 推奨環境

- **ストレージ容量**: 地図画像や写真の保存に応じた空き容量が必要（目安：数百MB以上）
- **メモリ**: スムーズな動作のため、十分な空きメモリを推奨
- **画面サイズ**: スマートフォン・タブレット・PCの各種画面サイズに対応

### 注意事項

- AI機能（画像判別・音声文字起こし等）を使用する場合は、Google Generative AI（Gemini）のAPIキー設定が必要です
- 大容量の画像ファイルを扱う場合、端末のメモリ容量によっては処理に時間がかかる場合があります
- オフライン環境でも基本的な機能は利用できますが、クラウド同期やAI機能はオンライン時のみ利用可能です
